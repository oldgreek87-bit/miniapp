# Функционал: Книга месяца и Журнал

## ✅ Реализовано

### Главная страница

1. **Книга месяца:**
   - Отображается по умолчанию при загрузке
   - Показывает изображение книги
   - Название и автор
   - Текст "регулярно читаем книги на английском" и "прокачиваем словарный запас"
   - Разворачивающаяся секция "О чём книга" с полным описанием
   - Подсказка о свайпе

2. **Журнал:**
   - Доступен через свайп влево по книге
   - Показывает изображение журнала
   - Название и краткое описание
   - Разворачивающаяся секция "О чём журнал" с полным описанием
   - Подсказка о свайпе обратно

3. **Свайп функционал:**
   - Свайп влево → показывает журнал
   - Свайп вправо → показывает книгу
   - Работает на touch устройствах

### Админ панель

1. **Вкладка "Контент":**
   - Форма для управления книгой месяца
   - Форма для управления журналом
   - Автоматическая загрузка текущих данных при открытии
   - Сохранение изменений

2. **Вкладка "Пользователи":**
   - Список всех пользователей
   - Информация о пользователях (имя, фото, username)
   - Управление подписками

## Структура данных

### База данных

**Таблица `book_of_month`:**
- `month` (INTEGER) - месяц (1-12)
- `year` (INTEGER) - год
- `title` (TEXT) - название
- `author` (TEXT) - автор
- `description` (TEXT) - полное описание
- `image_url` (TEXT) - URL изображения

**Таблица `monthly_magazine`:**
- `issue_number` (INTEGER) - номер выпуска
- `title` (TEXT) - название
- `short_description` (TEXT) - краткое описание
- `full_description` (TEXT) - полное описание
- `image_url` (TEXT) - URL изображения

## API Endpoints

### Получение данных

- `GET /api/book-of-month` - получить текущую книгу месяца
  - Возвращает книгу для текущего месяца/года
  - Если нет для текущего месяца, возвращает последнюю

- `GET /api/magazine/latest` - получить последний журнал
  - Возвращает журнал с наибольшим номером выпуска

### Обновление данных (требует админ токен)

- `POST /api/admin/book-of-month` - обновить книгу месяца
- `POST /api/admin/magazine` - обновить журнал

## Использование

### Для пользователей

1. Открыть приложение
2. Увидеть книгу месяца
3. Свайпнуть влево → увидеть журнал
4. Нажать "О чём книга/журнал" → прочитать описание

### Для админа

1. Открыть админ панель
2. Перейти на вкладку "Контент"
3. Заполнить формы
4. Сохранить
5. Изменения сразу отобразятся в приложении

## Особенности

- **Автоматическая загрузка** - данные загружаются при открытии страницы
- **Fallback значения** - если данных нет в БД, показываются дефолтные
- **Обработка ошибок изображений** - если изображение не загружается, используется дефолтное
- **Адаптивный дизайн** - работает на всех устройствах
