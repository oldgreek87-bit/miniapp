# Настройка контента (Книга месяца и Журнал)

## Первоначальная настройка

### Вариант 1: Через админ панель (рекомендуется)

1. Откройте админ панель: `https://your-app.vercel.app/admin.html?token=YOUR_ADMIN_TOKEN`
2. Перейдите на вкладку **"Контент"**
3. Заполните форму **"Книга месяца"**:
   - Месяц: 12 (Декабрь)
   - Год: 2025
   - Название: Двенадцать дней Рождества
   - Автор: Сьюзан Стоукс-Чепмен
   - Описание: (вставьте полный текст описания)
   - URL изображения: `book-of-month.png` (или полный URL)
4. Заполните форму **"Журнал"**:
   - Номер выпуска: 22
   - Название: Bookflix Monthly
   - Краткое описание: (базовое описание)
   - Полное описание: (полный текст о журнале)
   - URL изображения: `magazine-22.png` (или полный URL)
5. Нажмите **"Сохранить"** для каждой формы

### Вариант 2: Через скрипт (для разработки)

```bash
node init-december-data.js
```

## Управление контентом

### Обновление книги месяца

1. Откройте админ панель → вкладка "Контент"
2. Измените данные в форме "Книга месяца"
3. Нажмите "Сохранить книгу"

### Обновление журнала

1. Откройте админ панель → вкладка "Контент"
2. Измените данные в форме "Журнал"
3. Нажмите "Сохранить журнал"

## Изображения

### Локальные файлы

Если изображения находятся в корне проекта:
- `book-of-month.png` - для книги
- `magazine-22.png` - для журнала

В админ панели укажите просто имя файла: `book-of-month.png`

### Внешние URL

Если изображения загружены на внешний хостинг:
- Укажите полный URL: `https://example.com/images/book.jpg`

### Рекомендации

- Используйте изображения хорошего качества
- Оптимальный размер: 300-400px по ширине
- Формат: PNG или JPG
- Убедитесь, что файлы загружены в репозиторий или доступны по URL

## Структура данных

### Книга месяца

- **month** (1-12) - месяц
- **year** (2025) - год
- **title** - название книги
- **author** - автор
- **description** - полное описание (показывается в "О чём книга")
- **image_url** - URL или путь к изображению

### Журнал

- **issue_number** (22) - номер выпуска
- **title** - название журнала
- **short_description** - краткое описание (показывается всегда)
- **full_description** - полное описание (показывается в "О чём журнал")
- **image_url** - URL или путь к изображению

## Проверка

После сохранения:

1. Откройте главную страницу приложения
2. Проверьте, что книга месяца отображается корректно
3. Сделайте свайп - должен появиться журнал
4. Нажмите "О чём книга" - должно открыться описание
5. Нажмите "О чём журнал" - должно открыться полное описание

## API для программного управления

Вы также можете обновлять контент через API:

```bash
# Обновить книгу месяца
curl -X POST "https://your-app.vercel.app/api/admin/book-of-month" \
  -H "Content-Type: application/json" \
  -H "X-Admin-Token: YOUR_ADMIN_TOKEN" \
  -d '{
    "month": 12,
    "year": 2025,
    "title": "Двенадцать дней Рождества",
    "author": "Сьюзан Стоукс-Чепмен",
    "description": "Описание...",
    "image_url": "book-of-month.png"
  }'

# Обновить журнал
curl -X POST "https://your-app.vercel.app/api/admin/magazine" \
  -H "Content-Type: application/json" \
  -H "X-Admin-Token: YOUR_ADMIN_TOKEN" \
  -d '{
    "issue_number": 22,
    "title": "Bookflix Monthly",
    "short_description": "Краткое описание...",
    "full_description": "Полное описание...",
    "image_url": "magazine-22.png"
  }'
```
